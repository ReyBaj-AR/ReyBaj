Proyecto Mi Pan

Está pensado para pequeños panaderos artesanales que quieren comenzar a emprender.

Este software es una calculadora de porcentaje panadero, vos elegís la receta que querés hacer, ingresas la cantidad de harina que vas a utilizar para esa receta, y el sistema te devuelve las cantidades necesarias para el resto de los ingredientes.
A medida que vaya pasando el tiempo, voy a ir agregando mas recetas.
